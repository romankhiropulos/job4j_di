## Описание проекта.
Изучение DI и Spring

#### 1) Скачиваем проект job4j_di:
###### git clone путь_к_репозиторию

#### 2) Переходим в проект job4j_di:
###### cd job4j_di

#### 3) Запускаем команду сборки:
###### docker build -t job4j_di . 

#### 4) Запустим образ:
###### docker run job4j_di
